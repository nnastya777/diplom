# Руководство пользователя

## Система учета рабочего времени - Инструкция

### Вход в систему

1. Откройте систему в браузере
2. Введите email и пароль
3. Нажмите "Войти"

При первом входе вам придет письмо для подтверждения email.

### Для сотрудников

#### Отметка времени

**Отмечаем приход:**
1. Войдите в личный кабинет
2. В секции "Учет времени" нажмите "Отметить приход"
3. Текущее время будет зафиксировано

**Отмечаем уход:**
1. В конце рабочего дня нажмите "Отметить уход"
2. Система автоматически рассчитает отработанные часы

#### Подача заявки на отпуск

1. Перейдите в секцию "Отпуска и больничные"
2. Нажмите "Новая заявка"
3. Выберите тип (отпуск/больничный/командировка)
4. Укажите даты начала и окончания
5. Добавьте примечание (опционально)
6. Нажмите "Подать заявку"

Статус заявки можно отслеживать в этой же секции.

#### Просмотр статистики

В секции "Статистика" доступна информация:
- Отработанные часы за месяц
- Количество опозданий
- Дни отсутствия
- Баланс отпускных дней

### Для руководителей

#### Мониторинг команды

1. Перейдите в "Управление отделом"
2. Просмотрите список сотрудников
3. Проверьте статусы присутствия

#### Одобрение заявок

1. В секции "Заявки на рассмотрении" отображаются все новые заявки
2. Нажмите на заявку для просмотра деталей
3. Выберите "Одобрить" или "Отклонить"
4. Сотрудник получит уведомление

#### Контроль посещаемости

В разделе "Мониторинг посещаемости":
- Зеленый статус - присутствует
- Желтый - опоздание
- Красный - отсутствует

### Для администраторов

#### Управление пользователями

1. Откройте "Администрирование"
2. Перейдите в "Управление пользователями"
3. Для добавления пользователя нажмите "Добавить сотрудника"
4. Заполните форму и выберите роль

**Изменение роли:**
- employee - обычный сотрудник
- manager - руководитель отдела
- admin - администратор системы

#### Управление отделами

В секции "Управление отделами":
- Добавление новых отделов
- Изменение названий
- Удаление неиспользуемых

#### Системные настройки

**Рабочее время:**
- Начало рабочего дня
- Конец рабочего дня
- Допустимое опоздание

**Настройки уведомлений:**
- Email уведомления
- Push уведомления
- Частота напоминаний

### Формирование отчетов

#### Отчет по посещаемости

1. Откройте "Отчеты"
2. Выберите "Отчет по посещаемости"
3. Укажите период (даты начала и окончания)
4. Выберите отдел (для администраторов)
5. Нажмите "Сформировать отчет"
6. Экспортируйте в нужном формате (CSV/Excel/JSON)

#### Отчет по рабочим часам

Аналогично отчету по посещаемости, показывает:
- Обычные рабочие часы
- Сверхурочные часы
- Общее время работы

#### Расчет заработной платы

1. Выберите месяц для расчета
2. Укажите ставки (обычная/сверхурочная)
3. Система автоматически рассчитает ЗП на основе отработанных часов
4. Экспортируйте для бухгалтерии

### Частые вопросы

**Q: Забыл отметить уход, что делать?**
A: Обратитесь к руководителю для ручной корректировки записи.

**Q: Как изменить данные профиля?**
A: В личном кабинете перейдите в секцию "Профиль" и нажмите "Редактировать".

**Q: Не пришло письмо подтверждения**
A: Проверьте папку "Спам". Если письма нет, обратитесь к администратору.

**Q: Как узнать остаток отпускных дней?**
A: Информация доступна в секции "Статистика" личного кабинета.

**Q: Можно ли отменить заявку на отпуск?**
A: Да, пока заявка имеет статус "На рассмотрении", вы можете её отменить.

### Техническая поддержка

При возникновении проблем обращайтесь к администратору системы.
